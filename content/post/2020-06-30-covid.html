Covid Friday

El Covid Friday es el nombre al día sin IVA dado por Bloomberg,y es gracias a la forma aparente de como se manejo el distanciamiento social por parte de las grandes superficies en el país2

En este punto comparto la preocupación de varios medios internacionales dado que conoceremos los efectos directos de esta política a nivel de contagios hasta el nueve de Julio y con base a ello hay un problema menor sobre como separar los escenarios del aumento de los contagios gracias a esta implementación.

Por el momento lo que sí se puede hacer es evaluar cómo ha sido la evolución de este hecho a un nivel muy macro.

Antes de ver el texto, lo voy a limpiar y para ello cree la siguiente función.

limpiar_texto<-function(texto){
  texto <- gsub("(RT|via)((?:\\b\\W*@\\w+)+)", "", texto)
  texto <- gsub("@\\w+", "", texto)
  texto <-  gsub("\\bhttp[a-zA-Z0-9]*\\b", "", texto)
  texto <- gsub("[^a-zA-Z ]", "", texto)
  texto <- gsub("[[:punct:]]", "", texto)
  texto <- gsub("amp ", "", texto)
  texto <-  gsub("\\btco[a-zA-Z0-9]*\\b", "", texto)
  texto <- texto[!is.na(texto)]
  texto <- iconv(texto, 'UTF-8', 'ASCII')
  texto <- gsub("^\\s+|\\s+$", "", texto)
  texto <- tolower(texto)
  return(texto)
}
